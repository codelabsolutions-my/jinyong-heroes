# jinyong-heroes — Project Overview

> The **WHY / WHAT**. CLAUDE.md is the HOW. 系统与内容设计见 GAME_DESIGN.md。

## Vision

《金庸群侠传》(1996) 的现代化 Web 重制：保留原版"穿越武侠世界、集齐十四天书、
自由招募群侠"的开放式骨架，用现代游戏设计（任务日志、扩展剧情、门派声望、多结局）
重做体验。个人学习/同人项目，非商业，浏览器即开即玩。

## Scope — current milestone: M1 江湖骨架

（M0 已完成：占位地图上逐格移动 + 碰撞 + 摄像机跟随）

1. 至少 2 张地图，走到出入口能切换地图
2. NPC 摆放在地图上，走到面前按键触发对话框
3. 对话产出线索 → 任务日志界面可查看
4. 存档/读档（localStorage，带 version 的 GameState）
5. 以上全部有游戏逻辑层单测；浏览器实际玩通

## Out of scope (explicitly)

- 实时/动作战斗、多人、随机生成内容、配音
- 原版素材复用（美术/音乐全部自制、AI 生成或 CC0/CC-BY）
- 商业化（金庸改编权不在我们手上）
- M2 前不做战斗；M5 前不追求最终美术

## Roadmap

- M0 走起来 ✅ — 地图 + 逐格移动（2026-07）
- M1 江湖骨架 — 多地图、NPC 对话、任务日志、存档
- M2 第一场架 — 完整战棋战斗
- M3 第一部天书 — 一条完整小说线（射雕片段）
- M4 江湖成形 — 正邪值、门派声望、3 条小说线
- M5 纵向切片 — 开局到 1 个结局 + 像素素材替换占位块
