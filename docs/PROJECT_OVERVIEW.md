# jinyong-heroes — Project Overview

> The **WHY / WHAT**. CLAUDE.md is the HOW. 系统与内容设计见 GAME_DESIGN.md。

## Vision

《金庸群侠传》(1996) 的现代化 Web 重制：保留原版"穿越武侠世界、集齐十四天书、
自由招募群侠"的开放式骨架，用现代游戏设计（任务日志、扩展剧情、门派声望、多结局）
重做体验。个人学习/同人项目，非商业，浏览器即开即玩。

## Scope — current milestone: M3 第一部天书

（M0、M1、M2 已完成，见 Roadmap；M3 详细计划见 NEXT_STEPS.md §2）

一条完整的小说线（《射雕》第 1、3 章，见 STORY_BIBLE §2.9），验证"剧情链 + 战斗 +
奖励 + 天书"闭环：

1. 剧情事件链：声明式条件驱动的多步事件（对话→战斗→奖励→setFlag），跨章 flag 串联
2. 新地图（牛家村/华山）+ 新遭遇；战斗奖励接通（历练值/武学习得）
3. `books: string[]` 进 GameState（**bump SAVE_VERSION→2 + v1→v2 迁移 + 迁移单测**）
4. 第一部天书《射雕》入手；日志新增"天书"分册显示进度
5. 郭靖/黄蓉按 CHARACTERS §4 系数作剧情战友军登场（招募系统 M4 才上）
6. 以上有逻辑层单测 + e2e 实测从触发到拿到天书的完整流程

## Out of scope (explicitly)

- 实时/动作战斗、多人、随机生成内容、配音
- 原版素材复用（美术/音乐全部自制、AI 生成或 CC0/CC-BY）
- 商业化（金庸改编权不在我们手上）
- M4 前不做招募/正邪值/门派声望；M5 前不追求最终美术
