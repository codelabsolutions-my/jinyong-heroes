# jinyong-heroes — Project Overview

> The **WHY / WHAT**. CLAUDE.md is the HOW. 系统与内容设计见 GAME_DESIGN.md。

## Vision

《金庸群侠传》(1996) 的现代化 Web 重制：保留原版"穿越武侠世界、集齐十四天书、
自由招募群侠"的开放式骨架，用现代游戏设计（任务日志、扩展剧情、门派声望、多结局）
重做体验。个人学习/同人项目，非商业，浏览器即开即玩。

## Out of scope (explicitly)

- 实时/动作战斗、多人、随机生成内容、配音
- 原版素材复用（美术/音乐全部自制、AI 生成或 CC0/CC-BY）
- 商业化（金庸改编权不在我们手上）
- M4 前不做招募/正邪值/门派声望；M5 前不追求最终美术

## Pivot — 内容优先（2026-07，ADR #34）

M5 纵向切片跑通后，判断"引擎已够，缺的是**被设计过**的世界/剧情/NPC"（此前 42 张世界图是
生成的空壳、剧情靠 e2e 逼出来的）。决定**暂停堆功能，先把全 14 线的内容层设计成册**再逐线建：

- 生产桥文档：`WORLD_ATLAS.md`（60 图）·`ROSTER.md`·`CONTENT_FORMAT.md`·`ENDINGS.md`·`docs/lines/NN-*.md`（14 线 spec）。
- 生成的空壳世界已隔离（quarantine），照 atlas 逐张重画真图。
- 之后每条线 = 一个 workstream：读 `docs/lines/` spec → 建图/摆 NPC/写事件 → 单测+e2e。
