# STORY_BIBLE.md — 剧情总纲（十四天书全流程）

> 全部剧情内容的单一事实来源。写剧情的 session 从这里领任务，写完的内容不得与
> 本纲冲突；要改设定先改这里（同 PR）。细度约定：**本纲写到章节级**——每章的
> 地点、关键事件、战斗、抉择点；对话级细节在实施对应线时展开成 `src/data/` 数据。
>
> ⚠️ 所有剧情是基于金庸小说的**同人再创作**（非商业，见 CLAUDE.md §2.2），
> 章节流程是本项目原创设计，只参考原版游戏的"精神"（开放收集、自由顺序、正邪抉择），
> 不复刻其具体任务。人物性格必须忠于原著，原创配角不得喧宾夺主（GAME_DESIGN §3.1）。

## 1. 总体结构

### 1.1 骨架

```
开局：无名小村（已实现）→ 扫地老人点明主线：集齐十四天书方能归去
   ↓
中原大地图（枢纽）：各区域入口按难度层软性把关（低层区域怪弱，高层区域怪强）
   ↓
14 条小说线自由顺序推进，每条线走完最终章 → 获得该部天书
   ↓
集齐 14 部 → 终局事件 → 按正邪值/关键抉择进入 4 结局之一
```

### 1.2 难度分层（软性引导，不锁死——原版精神：去哪都行，打不打得过另说）

| 层  | 线                                               | 定位                         |
| --- | ------------------------------------------------ | ---------------------------- |
| T1  | 鸳鸯刀 · 白马啸西风 · 雪山飞狐                   | 短线（1-2 章），新手教学     |
| T2  | 碧血剑 · 书剑恩仇录 · 连城诀 · 飞狐外传 · 侠客行 | 中线（2-3 章）               |
| T3  | 射雕 · 神雕 · 倚天 · 天龙 · 笑傲 · 鹿鼎          | 长线（3-4 章），大战与大抉择 |

### 1.3 天书获取总规则

- 天书 = 该线**最终章的剧情奖励**，不可错过、不可重复（`grantBook` effect，M3 实现）。
- GameState 加 `books: string[]`（需 bump SAVE_VERSION，随 M3 一起做）。
- 日志新增"天书"分册：14 格进度，未获得的显示该线的入口线索（传闻导流）。

### 1.4 四结局矩阵（终局事件里判定，优先级从上到下）

| 结局         | 条件                                                                |
| ------------ | ------------------------------------------------------------------- |
| 侠之大者     | 14 书 + 侠值高 + 完成〔襄阳守城〕且〔选择留下〕→ 拒绝归去，镇守江湖 |
| 魔道称尊     | 14 书 + 邪值高 + 走完〔黑木崖夺位〕或〔光明顶邪线〕→ 君临魔道       |
| 逍遥归隐     | 14 书 + 任一伴侣羁绊满 + 正邪中庸 → 携手归隐，不问归途              |
| 归去（默认） | 14 书 → 回到现实世界                                                |

### 1.5 系统依赖图例（每条线标注需要的引擎能力 → 决定排期）

`战` 战棋战斗(M2) · `募` 招募入队(M4) · `正邪` 正邪值(M4) · `望` 门派声望(M4) ·
`谜` 解谜/机关 · `潜` 潜入 · `城` 大型守城/车轮战 · `互` 互斥招募/互斥抉择

## 2. 十四线章节流程

> 格式：主题一句话 → 章节 → 天书获取 → 可招募 → 正邪分支 → 跨线联动 → 系统依赖。
> 「抉择」= 影响正邪值或后续分支的选择点；「BOSS」= 强制战斗。

### 2.1 鸳鸯刀（T1，1 章）——推荐玩家的第一条线

- **主题**：喜剧护镖。太岳四侠劫镖闹笑话，鸳鸯刀里藏着"仁者无敌"的秘密。
- **第 1 章 · 双刀合璧**：官道遇太岳四侠劫镖（教学战，敌方极弱）→ 护送萧家镖车 →
  萧半和寿宴风波，BOSS 卓天雄 → 鸳鸯刀合璧现"仁者无敌"四字。
- **天书**：战后萧中慧相赠。
- **招募**：无（太岳四侠为固定笑点客串，多线彩蛋复用）。
- **正邪**：劫富济贫小抉择 ×1（放走/扭送太岳四侠）。
- **联动**：太岳四侠此后在各大城镇随机出现，卖破烂情报（传闻线索来源）。
- **系统**：`战`。**M3 后即可实施，是战斗系统的教学关。**

### 2.2 白马啸西风（T1，2 章）

- **主题**：回疆孤女李文秀与高昌迷宫——"那都是很好很好的，可我偏不喜欢"。
- **第 1 章 · 大漠白马**：回疆哈萨克部落，吕梁三杰追杀取地图，护住李文秀（BOSS）。
- **第 2 章 · 高昌迷宫**：进迷宫（机关+岔路解谜），发现"宝藏"只是中原日用品 →
  抉择：把真相告诉部落 / 保守秘密。
- **天书**：迷宫深处石匣。
- **招募**：李文秀（轻功高、攻击低的辅助型）。
- **联动**：完成后李文秀在结局〔逍遥归隐〕有专属送别台词。
- **系统**：`战` `谜`(迷宫地图=普通地图+机关格，M4 前用简化版) `募`。

### 2.3 雪山飞狐（T1，2 章）

- **主题**：玉笔峰上的罗生门——一桩百年恩怨，各人各说各话。
- **第 1 章 · 玉笔峰问案**：宝树和尚、平阿四、苗若兰各讲一段往事，证词互相矛盾 →
  **推理小游戏**：指认谁在说谎（答错无惩罚，答对拿额外线索）。
- **第 2 章 · 雪山藏宝**：苗人凤比武（BOSS，可"点到为止"或"下死手"——抉择）→ 藏宝洞。
- **天书**：藏宝洞内（拿刀谱宝藏与否是独立抉择：拿=邪+，封洞=侠+）。
- **招募**：无（苗人凤太强，设定为不出山）。
- **联动**：若已完成〔飞狐外传〕，胡斐在第 2 章登场代打苗人凤一战（羁绊剧情）。
- **系统**：`战` `谜` `正邪`。

### 2.4 飞狐外传（T2，3 章）

- **主题**：少年胡斐的侠义路——为素不相识的钟阿四一家讨公道。
- **第 1 章 · 商家堡**：目击凤天南强占民女案 → 抉择：出手/旁观（旁观直接 邪+，线暂停，
  需再访才重启）。
- **第 2 章 · 佛山追凶**：追凤天南，汤沛设局，BOSS 凤天南护卫队。
- **第 3 章 · 药王庄**：中毒 → 程灵素解毒（她的考验：三个药理问答）→ 抉择：
  劝胡斐放下屠刀 / 助他手刃凤天南（杀=快意但 邪+，劝=侠+）。
- **天书**：程灵素相赠。
- **招募**：胡斐（高攻近战）、程灵素（用毒/治疗，全游戏唯一治疗系队友——稀缺定位）。
- **联动**：→ 雪山飞狐（见 2.3）；程灵素在队时〔药王庄〕可反复买解毒剂。
- **系统**：`战` `募` `正邪`。

### 2.5 连城诀（T2，3 章）

- **主题**：人心比刀剑毒——连城宝藏照出众生贪相。
- **第 1 章 · 湘西冤狱**：狄云被万圭构陷 → 潜入万府取证（潜入玩法简化版：避开巡逻格）。
- **第 2 章 · 血刀凶威**：雪谷，BOSS 血刀老祖（高难，允许打不过跑路，线索提示练级）。
- **第 3 章 · 天宁寺宝藏**：宝藏现世，群豪疯抢 → **本线核心抉择**：碰宝藏（邪++，
  且触发"毒发"事件损 hp 上限）/ 只取天书离开（侠+）。
- **天书**：天宁寺佛像中。
- **招募**：狄云（大器晚成型：初始弱，羁绊事件后暴涨）、水笙。
- **系统**：`战` `募` `正邪` `潜`(简化)。

### 2.6 书剑恩仇录（T2，3 章）

- **主题**：红花会群雄营救文泰来。
- **第 1 章 · 铁胆庄**：文泰来被擒，红花会求助 → 抉择：帮红花会/告发（告发=邪线独有
  剧情，清廷赏金，望-红花会 敌对）。
- **第 2 章 · 六和塔营救**：潜入六和塔（潜入玩法：守卫视野格），救出文泰来，BOSS 张召重。
- **第 3 章 · 回疆风沙**：护送香香公主，狼群车轮战。
- **天书**：陈家洛以红花会信物相赠（邪线：从张召重尸身搜出）。
- **招募**：无尘道长（独臂快剑）、心砚（书童，彩蛋弱角）。
- **联动**：红花会声望在〔飞狐外传〕胡斐对话有变体。
- **系统**：`战` `募` `正邪` `望` `潜` `城`(狼群车轮战=多波次战斗)。

### 2.7 碧血剑（T2，3 章）

- **主题**：金蛇郎君的遗产与袁崇焕的遗志。
- **第 1 章 · 华山石洞**：山洞机关谜（金蛇秘笈+金蛇剑），学〔金蛇剑法〕（全玩家可学，
  本线福利）。
- **第 2 章 · 温家堡**：温氏五行阵（特殊战斗：击破阵眼单位阵法才解除）。
- **第 3 章 · 京城风云**：袁承志刺杀皇帝的计划 → 抉择：助他放下（侠+）/ 助他动手
  （剧情必然失败，邪+，袁承志退场不可招募）。
- **天书**：温家堡地窖。
- **招募**：袁承志（均衡型）、何铁手（用毒，邪值高才可招——邪线专属队友）。
- **系统**：`战` `募` `正邪` `谜` （阵法=battle 特殊胜利条件，resolve 需支持"击破特定单位"）。

### 2.8 侠客行（T2，2 章）

- **主题**：不识字的人参透了天下第一武学。
- **第 1 章 · 赏善罚恶令**：龙木二岛主邀"喝腊八粥"，武林闻风丧胆 → 随石破天上侠客岛
  （抉择：替人赴死接令=侠++ / 被迫上岛）。
- **第 2 章 · 石壁太玄经**：石壁二十四室参悟——**反向解谜**：房间里给出各家注解选项，
  全部选"不看注解，只看笔画"才能悟出（选注解越多越走火，提示越明显）。
- **天书**：参透石壁即得（《太玄经》与天书合一，本线同时送顶级内功〔太玄功〕——
  全游戏数值天花板，放在解谜线而非战斗线是刻意设计：奖励智取）。
- **招募**：石破天（憨傻高质量肉盾，对话全是错位笑点）。
- **系统**：`战` `谜` `募`。

### 2.9 射雕英雄传（T3，3 章）——**M3 首发线**

- **主题**：侠之大者，为国为民。
- **第 1 章 · 风雪惊变**：张家口遇乞丐装黄蓉（不揭穿=羁绊+），牛家村曲三酒馆旧案，
  BOSS 黄河四鬼。
- **第 2 章 · 桃花岛**：黄药师三试（乐律谜题 / 奇门阵走位 / 战郭靖——点到为止）。
- **第 3 章 · 华山之巅**：华山论剑旁观 → 欧阳锋抢《九阴真经》，BOSS 欧阳锋
  （设计为"打不过也能过"：坚持 N 回合即触发洪七公救场，打赢有额外奖励）。
- **天书**：洪七公以"侠义为先"相授。
- **招募**：郭靖（重装）、黄蓉（军师型：战斗中"计策"技能——降敌速/乱敌阵）。
- **正邪**：本线是"侠"字招牌线，多个小抉择全部偏正。
- **联动**：→ 神雕前置（完成后神雕线中郭靖夫妇登场）；〔降龙十八掌〕需郭靖羁绊满+侠值高。
- **系统**：`战` `募` `谜`。**M3 只做 1、3 章（桃花岛章 M4 补），验证"剧情链+战斗+天书奖励"闭环。**

### 2.10 神雕侠侣（T3，3 章）——前置：射雕完成

- **主题**：问世间情为何物。
- **第 1 章 · 活死人墓**：全真教冲突 → 古墓，杨过小龙女初见（玩家撞破礼教非议 →
  抉择：世俗劝分 / 支持二人（支持=羁绊线开启））。
- **第 2 章 · 绝情谷**：情花毒，裘千尺枣核钉（BOSS），断肠草抉择（救杨过须玩家
  跳崖采药——假死惊吓桥段）。
- **第 3 章 · 襄阳大战**：**守城战**：多波蒙古兵+BOSS 金轮法王，郭靖黄蓉参战
  （射雕队友自动加入本战）。
- **天书**：战后郭靖授"侠之大者"训 → 天书。此战胜利是结局〔侠之大者〕的必要 flag。
- **招募**：杨过（爆发型，独臂后更强——招募时间点影响形态，彩蛋）、小龙女（双剑合璧：
  与杨过同场时二人全属性 +25%，全游戏唯一双人 buff）。
- **系统**：`战` `募` `城` `互`(杨过形态)。

### 2.11 倚天屠龙记（T3，4 章）

- **主题**：屠龙倚天，号令天下——正邪之辨最模糊的一条线。
- **第 1 章 · 蝴蝶谷**：胡青牛医仙的考验，张无忌身世。
- **第 2 章 · 光明顶**：**本线核心**：六大派围攻 → **选边**：助明教（世俗眼中邪+，
  实际剧情偏侠）/ 助六大派（望+各派，但目睹灭绝滥杀 → 后续反转）。连续 3 场战斗。
- **第 3 章 · 万安寺**：救被囚群雄（限时救援：回合数限制内破塔）。
- **第 4 章 · 屠狮大会**：谢逊与屠龙刀真相，BOSS 成昆（多阶段）。
- **天书**：屠龙刀中（"武穆遗书"位置放天书——同人改编点，需在日志里自嘲一句）。
- **招募**：张无忌（全能，双系武学）、赵敏 / 周芷若（**互斥**：光明顶站边+后续抉择决定
  谁可招募，另一人成为后期 BOSS——本项目最大的分支资产）。
- **系统**：`战` `募` `正邪` `望` `互` `城`(连续战) + 限时胜利条件。

### 2.12 天龙八部（T3，4 章）

- **主题**：无人不冤，有情皆孽。
- **第 1 章 · 大理无量山**：段誉坠崖学北冥/凌波（玩家旁观获〔凌波微步〕学习资格）。
- **第 2 章 · 珍珑棋局**：**解谜名场面**：棋局选项全是死路，唯"自填一子闭眼胡下"
  （虚竹解法）可破——与侠客行石壁呼应的"放下执念"主题。虚竹得逍遥传承。
- **第 3 章 · 聚贤庄**：乔峰契丹身世曝光 → **全场大战**：群雄车轮战 → 抉择：
  与乔峰并肩（望-中原各派，侠+"虽千万人吾往矣"）/ 袖手旁观。
- **第 4 章 · 雁门关**：辽宋对峙，乔峰之抉择（剧情不可改变结局——尊重原著悲剧内核，
  玩家只能见证；见证后侠值大幅+）。
- **天书**：乔峰遗赠（并肩线）/ 扫地僧转交（旁观线，藏经阁事件）。
- **招募**：段誉（游走控制）、虚竹（内力肉盾）；乔峰**不可招募但在第 3 章并肩作战**
  （原著地位特殊，做成"限定同行"比常驻队友更贵重）。
- **系统**：`战` `募` `谜` `城` `望`。

### 2.13 笑傲江湖（T3，4 章）

- **主题**：有人的地方就有江湖——伪君子与真小人。
- **第 1 章 · 衡山金盆洗手**：目击刘正风灭门（嵩山派执法）→ 埋"正派未必正"的钩子。
- **第 2 章 · 华山思过崖**：令狐冲被罚 → 风清扬授剑（玩家满足条件可学〔独孤九剑〕：
  需悟性事件 ×3——分散在前三章的隐藏对话）。
- **第 3 章 · 五霸冈**：圣姑任盈盈与群豪，绿竹巷琴箫（羁绊事件）。
- **第 4 章 · 黑木崖**：**正线**：助令狐冲救任盈盈，BOSS 东方不败（全游戏单体最强，
  速度碾压——设计为必须用地形/人数优势）；**邪线**：投效日月神教，替东方不败清理门户
  （结局〔魔道称尊〕的主要入口）。
- **天书**：黑木崖秘库。
- **招募**：令狐冲（剑客暴击流）、任盈盈（辅助）；二人同队触发羁绊事件链（GAME_DESIGN
  §3.1 羁绊系统的标杆实现）。
- **系统**：`战` `募` `正邪` `望` `互`(正邪线互斥)。

### 2.14 鹿鼎记（T3，3 章）——反武侠喜剧线，刻意放最后实施

- **主题**：不会武功的人如何在武林活得最好。
- **第 1 章 · 京城**：遇韦小宝，卷入宫廷斗（**嘴炮玩法**：对话选择代替战斗过关——
  选错才打）。

## 5. 写作规范（给内容 session）

1. 人物说话必须像原著那个人——动笔前重读该角色原著名场面。
2. 抉择必须两难或有代价，不做"明显正确答案"；正邪偏移量在实施时统一定表，不逐条拍脑袋。
3. 悲剧内核不改写（乔峰、程灵素等）——玩家可以见证、不能扭转；同人的边界是"补白"，
   不是"翻案"。
4. 每章 1 个名场面 + 1 个战斗/解谜 + 至少 1 个抉择，超过 4 场战斗就拆章。
5. 所有新内容遵守 NEXT_STEPS §5 的数据流程；变体防线索丢失规则（content.test.ts）适用于
   天书：`grantBook` 永远放在无变体遮蔽的主链事件上。
